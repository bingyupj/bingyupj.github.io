Hugo安装并发布到Github

Posted on Feb 23, 2018
Hugo 是什么？ The world’s fastest framework for building websites. 官方说是世界上最快的建立静态网站的工具。 相类似的还有hexo,是node js 写的，hugo 是用 go 写的。 以前玩过hexo,现在还是用下hugo吧。 一：下载 官网下载地址 1）windows 环境 直接从 [releases地址](https://github.com/gohugoio/hugo/releases) 上下载相应的安装包。 下好后，是个hugo.exe文件，把该目录直接丢到环境变量里面去，方便操作。 2） mac 环境 用 brew install hugo 或者同样的从上面地址下载即可。 查看版本 ：hugo version 二 生成站点 hugo new site blog 创建的站点文件目录说明： |- archetypes ：存放default.md，头文件格式 |- content ：content目录存放博客文章（.markdown/.md文件） |- data ：存放自定义模版，导入的toml文件（或json，yaml） |- layouts ：layouts目录存放的是网站的模板文件 |- static ：static目录存放js/css/img等静态资源 |- config.toml ：config.toml是网站的配置文件 当前网站是没有任何内容的，需要下载个主题跑起来才有内容。 三 下载主题 Hugo官方主题 、Hugo官方主题GitHub 从官网上选好了一个主题如：https://themes.gohugo.io/vienna/, 根据指引进行安装，cd到 themes 目录下： git clone https://github.
